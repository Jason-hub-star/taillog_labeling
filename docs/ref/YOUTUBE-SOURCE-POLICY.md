# YouTube Source Policy — SSOT

> YouTube 영상 수집 기준·검색어·품질 필터·보존 정책 정의.

---

## 수집 전략

- **키워드**: 불필요. 일상 강아지 영상 우선 (AP-10K 모델이 일반 영상 처리 가능)
- **초기 검색어**:
  ```
  KO: "강아지 일상", "강아지 홈캠", "강아지 브이로그", "멍멍이 일상", "강아지 훈련"
  EN: "dog daily routine", "dog home camera", "dog vlog", "puppy daily life", "dog training session"
  ```
- **채널 우선순위**:
  1. 반려견 훈련사 채널 (ABC 행동 맥락 풍부)
  2. 반려견 일상 채널 (자연스러운 행동 분포)
  3. 펫샵/보호소 채널 (다양한 견종)

## 영상 품질 기준 (수집 단계 필터)

| 기준 | 값 | 필터 단계 |
|------|----|---------:|
| 최소 해상도 | 480p | collector |
| 최소 길이 | 10초 | collector |
| 최대 길이 | 10분 | collector |
| 강아지 비율 | 바운딩박스 ≥ 화면 5% | pose_extractor |
| 최소 FPS | 10fps (원본) | collector |

## 피해야 할 영상

- 편집 과다 유튜브 예능 (갑작스러운 cut이 많아 포즈 추출 불안정)
- CGI / 애니메이션
- 강아지 얼굴 클로즈업만 (전신 포즈 필요)
- 흑백/저화질 (480p 미만)

## 영상 보존 정책

| 상태 | 보존 기간 | 처리 |
|------|---------|------|
| 다운로드 완료, 라벨링 미완료 | 최대 14일 | 자동 삭제 스케줄 |
| 라벨링 완료 (synced) | 7일 추가 | 검수 후 삭제 |
| 라벨링 완료 (human_review) | 30일 | 검수 완료 후 삭제 |
| 라벨링 실패 | 3일 | 자동 삭제 |

> **원본 영상은 절대 Supabase/외부 서버에 저장하지 않는다.**
> 포즈 키포인트(숫자 좌표)만 Supabase로 전송됨.

## dedupe 정책

- **기준**: `dedupe_key = SHA256(youtube_url + frame_range)`
- **같은 URL 재수집 허용**: frame_range가 다르면 다른 라벨 허용
- **채널 중복**: 같은 채널에서 동일 행동 패턴 반복 → 허용 (실제 행동 분포 반영)

## Rate Limiting 정책

- **다운로드 간격**: 영상 1개당 최소 2초 대기
- **일일 최대**: YouTube 계정당 500건 (yt-dlp 기본 제한 회피)
- **IP 차단 시**: 자동 대기 30분 → retry
- **연속 차단 3회**: 수집 중단 + watchdog 알림

## 법적 고려사항

- 영상은 모델 학습용 로컬 처리 후 폐기 (공개 배포 없음)
- 추출된 키포인트 좌표는 개인정보가 아님 (숫자만)
- YouTube ToS: 개인 연구/비상업 목적 내 사용 (상업 배포 시 재검토 필요)
